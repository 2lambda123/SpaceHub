Detailed Description

template<typename Derived>
class space::StepController< Derived >

Abstract class of step size controller. A class implements(partly/fully) the interfaces of this class via CRTP idiom can be used cross the system as an implementation of the concept StepController. The step size controller provides the interface to estimate the next step size with current step size and errors in previous steps.

DerivedThe implement class in CRTP idiom.
