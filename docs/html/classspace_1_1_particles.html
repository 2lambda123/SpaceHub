Detailed Description

template<typename Derived>
class space::Particles< Derived >

Abstract class of a 'Structure of Array(SoA)' particle set. A class implements(partly/fully) the interfaces of this class via CRTP idiom can be used cross the system as an implementation of the concept Particles.

Template Parameters
DerivedThe implement class in CRTP idiom.
