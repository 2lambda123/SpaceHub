Detailed Description

template<typename Derived>
class space::Particles< Derived >

Abstract class of a 'Structure of Array(SoA)' kind SoA particle set. A class implements(partly/fully) the interfaces of this class via CRTP idioms can be used cross the system as an implementation of the concept Particles.

DerivedThe implement class in CRTP idioms.
