Detailed Description

template<typename Derived>
class space::ParticleSystem< Derived >

Abstract class of Particle System. A class implements(partly/fully) the interfaces of this class via CRTP idiom can be used cross the system as an implementation of the concept ParticleSystem. The particle System integrates the data set of a particle set and provides interface to evolve the internal particles.

Template Parameters
DerivedThe implement class in CRTP idiom.
