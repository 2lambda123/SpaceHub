template<typename Particles, typename Interactions, ReguType RegType>

Regularized particle System.

Regularied particle system. See details Explicit Symplectic Algorithms For Time‐Transformed Hamiltonians , A Class of Symplectic Integrators with Adaptive Time Step for Separable Hamiltonian Systems and A Time-Transformed Leapfrog Scheme.
