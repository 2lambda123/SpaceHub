template<typename Real, template< class... > class TContainer = std::vector>
struct space::Types< Real, TContainer >

Type system that is used across the Space Hub system. This type class provide all basic type i.e Scalar, Vector, ScalarArray, Coord and etc,.

Template Parameters
RealThe basic scalar type, i.e float, double, etc.
TContainerThe container type, i.e std::vector. Use std::vector as default.
