Getting started

Prerequisite

SpaceHub is written based on the morden-ish C++ standard c++17. To make the code compiled, the compiler on the target machine need to give a full support on at least c++17 or later. The lowest compiler version for different OS is listed below.

Operating SystemCompilerLowest Version requiredVersion recommended
LinuxGCC79
Linux/MacOSIntel C++19.0119.01
MacOsClang48
MacOsApple Clangsupportsupport
WindowsMSVC19.1419.14
WindowsGCC via CygWin/MinGw79 if supported

  • For Linux/MacOS user, check the version of your GCC in terminal,

    > g++ -v

    or check the version of the inte++ via,

    > icpc -v

    If your compiler version doesn't satisfy the lowest version required by SpaceHub, you need to update/install the new compiler.

    You can install the new GCC from scratch by executing the following commands by sequence,

    > wget https://ftp.gnu.org/gnu/gcc/gcc-9.1.0/gcc-9.1.0.tar.xz
> tar xf gcc-9.1.0.tar.xz
> cd gcc-9.1.0
> ./contrib/download_prerequisites
> mkdir build
> cd build
> ../configure --prefix=$HOME/GCC-9.1.0 --enable-languages=c,c++ --disable-multilib
> make -j 8
> make install

    after the installation, add the new g++ executable to your environment variable PATH by adding the following lines into your .bashrc(Linux)/.bash_profile(MacOS) under your directory ~,

    export PATH=$PATH:$HOME/GCC-9.1.0/bin
export LD_LIBRARY_PATH=$HOME/GCC-9.1.0/lib
export LD_LIBRARY_PATH=$HOME/GCC-9.1.0/lib64

    and source it

    > source ~/.bashrc

    or

    > source ~/.bash_profile

    then, check the GCC version again via

    > g++ -v

  • For Windows user, if you have installed Microsoft Visual Studio, check the version of your MSVC in cmd terminal,

    path_to_your_installed_dir\Microsoft Visual Studio xxx\VC > cl

    or just open your Microsoft Visual Studio, check the version information through about or help.

    If your MSVC doesn't satisfy the lowest version required by SpaceHub, update Microsoft Visual Studio to the newest version. If Microsoft Visual Studio is too heavy for you, you can also install the GCC on Windows via CygWin or MinGw.

  • For cluster user, if the cluster has build system like EasyBuild, check if any gcc module with version later than 7 is available via

    > module avail

    If any GCC module is available, e.g gcc/9.2.0, you can use

    > module load gcc/9.2.0

    to load the GCC-9.2.0. Otherwise, you need to install the GCC under your own directories without root privilege through the method above for linux user.

Install

SpaceHub is a header only code(In c++20, we will make into Modules). No installation is required, you only need to #include"path_to_spacehub/src/spaceHub.hpp" when writing your own code.

  • For Linux/MacOS user, open the terminal and cd to a place where you want to install the SpaceHub,

    > git clone https://github.com/YihanWangAstro/SpaceHub.git

    That it.

  • For Windows user, download the repository of SpaceHub and unpack it.

    SpaceHub
    download
