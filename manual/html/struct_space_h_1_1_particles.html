Detailed Description

template<typename Dtype, size_t ArraySize, bool IsVelDep>
struct SpaceH::Particles< Dtype, ArraySize, IsVelDep >

Basic particles group, wrapper on VelIndepParticles and VelDepParticles.

Template Parameters
DtypeType of scalar. e.g., float, double, kahanNumber...
ArraySizeThe size of the arrays in whole system. SpaceH::DYNAMICAL for dynamical array.
IsVelDepTemplate parameters to determine if the particles are velocity dependent.
